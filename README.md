### 快速入门

使用以下命令快速搭建（node>18）

```bash
npm install
```

```bash
npm run dev
```

```bash
npm run build
```

安装[NovelHub Admin](https://github.com/zhanghua12345/NovelHub)后，运行 `npm install` 安装依赖，然后运行 `npm run dev` 启动本地开发服务器。
